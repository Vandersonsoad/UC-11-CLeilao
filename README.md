# Sistema de Leilões (UC15)

## 📝 Descrição do Projeto
Aplicação desenvolvida para gestão e controle de leilões, permitindo a listagem, cadastro e monitoramento de lotes e lances de forma simples e intuitiva. O projeto faz parte das atividades práticas da Unidade Curricular 15.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Java (Swing para Interface Gráfica)
- **IDE:** NetBeans IDE
- **Banco de Dados:** MySQL
- **Controle de Versão:** Git & GitHub
